# This Week

## Focus

Create a tiny local StarHub-inspired test target and prove it can be checked with Playwright.

## Outcomes

- [x] Pick one page or journey from `www.starhub.com` to model first.
- [x] Scaffold a local site that can run in the browser.
- [x] Add one Playwright smoke test that verifies the page loads and a key CTA or navigation element works.
- [x] Confirm the suite passes in Chromium.

## Notes

- Start with a narrow demo. The first win is not a perfect replica; it is a reliable loop from local page to automated test result.
- Keep the replica free of internal StarHub code, data, and assets unless you have explicit permission to use them.
